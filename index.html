Global strategy to deal with bugs

Analyze the problem

  • Point out the steps to reproduce the problem.
  • Go to the basics
  • List the possible origins of the problem
  • Check the possible solutons
  • Choose long term solutions over quick fixes (if it is possible)
  • Think what is the underlying problem that is causing the bug

Global strategy to deal with bugs

Fix the problem

  • Ensure bug is fixed and double check there are no side effects
  • If the bug is too hard to fix, it would be good idea to refactor the feature?
  • Cover the bug with a test: Behat / phpunit / cypress / nightwatch...
